# 数据库设计与数据字典

## 1. 核心表

### users

`id`、`email`、`password_hash`、`nickname`、`role(student|teacher|admin)`、`status`、`teacher_subjects_json`、`grade_code`、`created_at`。历史 `user`→`student`。

### sessions

`token` 主键、`user_id`、`created_at`。首期 token 会话，生产可替换 JWT/Redis。

### generation_jobs

| 字段 | 类型 | 说明 |
|---|---|---|
| id | varchar(64) | Job 主键 |
| user_id | varchar(64) | 创建者 |
| status | varchar(32) | queued/running/succeeded/failed/cancelled |
| output_profile | varchar(64) | 六类输出档位 |
| input_json | JSON | 输入快照，不存 API Key |
| progress | int | 0-100 |
| current_stage | varchar(64) | 当前阶段 |
| work_dir | varchar(512) | 隔离工作目录 |
| video_url/cover_url | varchar(512) | 快速访问字段 |
| error_message | text | 最后一次失败原因 |
| created_at/updated_at/started_at/finished_at | datetime | 生命周期时间 |

### courses

课程元数据、审核状态、视频/封面 URL、来源 Job 和播放统计。

### course_assets

| 字段 | 类型 | 说明 |
|---|---|---|
| id | varchar(64) | 资产主键 |
| job_id | varchar(64) | 来源任务 |
| course_id | varchar(64) | 可空，课程创建后回填 |
| asset_type | varchar(32) | video_mp4/cover_png/infographic_png/diagram_png/storyboard_json/prompt_txt/metadata_json |
| path | varchar(512) | 服务端相对 URL |
| mime_type | varchar(128) | MIME |
| size_bytes | bigint | 文件大小 |
| metadata_json | JSON | provider、prompt hash、尺寸等 |
| created_at | datetime | 创建时间 |

## 2. 索引与约束

- `users.email` 唯一。
- `generation_jobs(status, created_at)` 支持队列扫描。
- `courses(publish_status, visibility, created_at)` 支持广场查询。
- `course_assets(job_id, asset_type)` 支持任务资产列表。


## subjects（学科）

| 字段 | 说明 |
|------|------|
| code | 学科编码 |
| name | 名称 |
| sort_order | 排序 |
| enabled | 是否启用 |

## knowledge_points（知识点）

| 字段 | 说明 |
|------|------|
| id | 知识点 ID |
| subject_code | 学科 |
| grade_code | 年级 |
| chapter | 章节 |
| topic | 知识点 |
| summary | 摘要 |
| keywords_json | 关键词 JSON 数组 |
| learning_goals_json | 学习目标 JSON 数组 |
| animation_pack | 动画包 |
| pack_key | 同步幂等键 |
| sort_order | 排序 |
| enabled | 是否启用 |
| source | 来源标记 |

年级取值：`grade1`…`grade12`。
