# RBAC / 审核回归清单（P4）

## 一键执行

```bash
npm run test:p4
```

## 覆盖项

1. 历史 `user` → `student` 迁移后可登录
2. 禁止公开注册 `admin`
3. 学生可创建全部 `OUTPUT_PROFILES` 任务
4. 教师待审仅本学科学生课
5. 跨学科详情/审核统一 **404**
6. 教师作品教师不可审（404），管理员可审
7. 无对口教师学生课 `needsAdminFallback=true`
8. 驳回意见回写 `GET /me/courses.latestReview`
9. 教师自改授课学科须 ≥1
10. 教师访问 `/admin/users` 为 403
